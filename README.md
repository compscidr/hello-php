# hello-php
Minimal setup with:
- composer
- php unit
- code cov
- gh actions
- secrets via .env

## local setup:
- requires php >= 8.1
- requires `php-dev`, `php-pear`
- requires pcov: `sudo pecl install pcov`
- copy the `sample.env` to `.env`
